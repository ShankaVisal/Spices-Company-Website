# **App Name**: Spiceverse

## Core Features:

- Product Catalog: Display the available spices products with images, descriptions and prices. The product data will be pulled from JSON files to allow for easy updates.
- Language Selection: Implement language toggle between English and Sinhala.
- WhatsApp Order Tool: Provide an option for customers to place orders via WhatsApp. Generate a pre-filled WhatsApp message based on the selected products.  Use the word "tool"
- Recipe Suggestion Tool: AI-powered tool to suggest recipes using the available spices. Suggest multiple options from a third party API, linking to full recipes.

## Style Guidelines:

- Primary color: Saffron (#F4C430) to evoke warmth, spice, and Sri Lankan heritage.
- Background color: Light beige (#F5F5DC) providing a neutral and clean backdrop to make product images stand out.
- Accent color: Burnt Sienna (#E9724C) for CTAs, highlights, and interactive elements, complementing the primary color and drawing the eye.
- Body text: 'PT Sans', a humanist sans-serif suitable for body text
- Headline font: 'Playfair', a modern serif, similar to Didot with geometric and high-contrast lines.
- Use a set of consistent and minimalist icons representing spices and product features.
- Modern, clean layout with ample white space to highlight product imagery. Grid-based design for product listing to showcase it in orderly way.
